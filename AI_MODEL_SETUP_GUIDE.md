# AI Model Setup Guide

## Overview

This guide helps you download and configure AI models for speech recognition and transcription features.

---

## Quick Start

The app works **without** any model files - AI features will simply be disabled until you add models. No errors or crashes!

---

## Speech Recognition Models

The app supports Whisper models for speech-to-text functionality:

### Whisper Tiny (Recommended for most users)
- **Size**: ~75 MB
- **Speed**: Fast on CPU
- **Languages**: 99 languages supported
- **Quality**: Good for most use cases

### Whisper Base (Balanced)
- **Size**: ~142 MB  
- **Speed**: Moderate
- **Languages**: 99 languages supported
- **Quality**: Better accuracy than Tiny

### Whisper Small (High Accuracy)
- **Size**: ~466 MB
- **Speed**: Slower but more accurate
- **Languages**: 99 languages supported
- **Quality**: Best for transcription

---

## Model Download

Models are downloaded automatically through the AI Models screen in the app:

1. Go to Settings → AI Models
2. Choose one speech recognition model
3. Tap "Download" to get the model
4. Wait for download to complete

---

## What Happens Without Models?

The app handles missing models gracefully:

1. ✓ App launches normally
2. ✓ Video playback works fine
3. ✓ All other features functional
4. ✗ Speech features are disabled
5. ℹ️ Shows message: "AI model not found. Download from AI Models screen"

**No crashes, no errors - just disabled features!**

---

## Performance Tips

1. **Start with Whisper Tiny** - test if quality is acceptable
2. **Use Base for better accuracy** - moderate size increase
3. **Use Small only if needed** - large file but best quality
4. **Models are shared** - one download works for all speech features
