🏗️  ARCHITECTURE
────────────────────────────────────────────────────────────────────────────

Audio Flow:
┌─────────────────────────────────────────────────────────────┐
│ Android playback capture (MediaProjection prompt)            │
│    ↓                                                        │
│ EventChannel stream → WhisperTranscriptionService           │
│    ↓                                                        │
│ Whisper TFLite Interpreter (Tiny/Base/Small)                │
│    ↓                                                        │
│ On-device decoding → transcript text                        │
│    ↓                                                        │
│ Optional ML Kit translation (offline packs)                 │
│    ↓                                                        │
│ Enhanced Video Player / Settings overlay                    │
└─────────────────────────────────────────────────────────────┘

Data Flow:
┌──────────────────────────────────────────┐
│ Settings Screen                          │
│  • Enable/Disable Transcription          │
│  • Select Source Language                │
│  • Enable/Disable Translation            │
│  • Select Target Language                │
│  • TTS Options                           │
│  • Export/Clear Options                  │
│         ↓                                 │
│ SharedPreferences (Settings Storage)     │
│         ↓                                 │
│ WhisperTranscriptionService              │
│ (handles transcription + translation)    │
│         ↓                                 │
│ Enhanced Video Player (Display)          │
└──────────────────────────────────────────┘
